# Smart Clinic Management System - User Stories

## 👨‍⚕️ Doctor User Stories

1. As a Doctor, I want to log in securely so that I can manage my appointments.
2. As a Doctor, I want to view my scheduled appointments so that I can prepare for consultations.
3. As a Doctor, I want to update appointment status so that patient records remain accurate.
4. As a Doctor, I want to view patient details so that I can provide proper treatment.

---

## 🧑‍🤝‍🧑 Patient User Stories

1. As a Patient, I want to register an account so that I can book appointments.
2. As a Patient, I want to log in securely so that I can access my medical information.
3. As a Patient, I want to book an appointment with a doctor so that I can receive treatment.
4. As a Patient, I want to view my appointment history so that I can track my visits.

---

## 👨‍💼 Admin User Stories

1. As an Admin, I want to manage doctor accounts so that the system stays updated.
2. As an Admin, I want to manage patient records so that data remains accurate.
3. As an Admin, I want to view all appointments so that I can monitor clinic operations.
4. As an Admin, I want to remove inactive users so that the system remains clean.
